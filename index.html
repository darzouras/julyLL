Typography

(For devs AND designers!)

5 Keys to Accessible Web Typography

  1. Set base font in relative units
  2. Check color and then contrast
  3. Use highly legible fonts
  4. Shape paragraphs well
  5. Use correct heading levels

1. Set base font sizes in relative units

							
								:root {
									font-size: 62.5%; /* (62.5/100) * 16px = 10px */
								}

								.font-size--small {
									font-size: 1.4rem; /* 14px */
								}

								.font-size--default {
									font-size: 1.6rem; /* 16px */
								}

								.font-size--large {
									font-size: 2.4rem; /* 24px */
								}

(Accessible Font Sizing Explained)

2. Check color and then contrast
